🔧 HOW EXCEPTION HANDLING WORKS NOW:

Request Flow:
  1. Controller receives @RequestBody with @Valid
  2. If DTO validation fails → MethodArgumentNotValidException
  3. GlobalExceptionHandler catches it
  4. Returns 400 with field-level error messages

  OR

  1. Business logic throws custom exception
  2. GlobalExceptionHandler catches it (ResourceNotFoundException, etc.)
  3. Returns appropriate HTTP status with error message
  4. Client receives standardized ApiResponse format
